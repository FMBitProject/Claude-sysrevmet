## 📁 How to Use Each Database

### 1. DOAJ (Directory of Open Access Journals)
```
Best for: Q3-Q5 open access journals
Coverage: 18,000+ journals, 100% open access
Access: Free, no API key needed

Usage:
- Check "DOAJ" in database list
- Search will include all OA journals
- Results have direct full-text links
```

### 2. SciELO (Scientific Electronic Library Online)
```
Best for: Latin American, Spanish, Portuguese journals
Coverage: 1,500+ journals from 16 countries
Access: Free, no API key needed

Usage:
- Check "SciELO" in database list
- Great for tropical medicine, public health
- Multilingual (English, Spanish, Portuguese)
```

### 3. PubMed Central (PMC)
```
Best for: Full-text biomedical articles
Coverage: 8+ million full-text articles
Access: Free, NIH-funded

Usage:
- Check "PubMed Central" (different from PubMed!)
- Get full-text, not just citations
- Better for AI data extraction
```

### 4. medRxiv / bioRxiv
```
Best for: Latest preprints (before peer review)
Coverage: Health sciences / Biology preprints
Access: Free, immediate

Usage:
- Check "medRxiv" or "bioRxiv"
- Get cutting-edge research
- Note: Not yet peer-reviewed!
```

### 5. ClinicalTrials.gov
```
Best for: Registered clinical trials
Coverage: 400,000+ trials worldwide
Access: Free, NIH-funded

Usage:
- Check "ClinicalTrials.gov"
- Find ongoing/completed trials
- Get trial protocols and results
```

### 6. J-STage (Japan Science and Technology)
```
Best for: Asian journals, especially Japanese
Coverage: 3,000+ journals
Access: Free, most full-text

Usage:
- Check "J-STage"
- Good for Asian population studies
- Many English articles
```

### 7. BASE (Bielefeld Academic Search Engine)
```
Best for: Multidisciplinary search
Coverage: 300+ million documents
Access: Free, one of largest search engines

Usage:
- Check "BASE"
- Broad multidisciplinary coverage
- Good for grey literature
```

### 8. CORE
```
Best for: Aggregated open access papers
Coverage: 135+ million OA papers
Access: Free, largest OA aggregator

Usage:
- Check "CORE"
- Searches repositories worldwide
- 100% open access
```

### 9. Epistemonikos
```
Best for: Systematic reviews and meta-analyses
Coverage: Largest database of systematic reviews
Access: Free, specialized

Usage:
- Check "Epistemonikos"
- Find existing systematic reviews
- Avoid duplication
```

```

```
